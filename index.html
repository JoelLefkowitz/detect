Detect  1.0.1
A single header library to detect the OS at compile time.
Usage

Using compiler predefined macros we can detect the operating system at compile time. These are aliased so that we can conditionally compile other sections in a readable way.

#include "detect.hpp"
#if PLATFORM_DETECTED_OS == PLATFORM_LINUX
...
#endif

The platform macros exported are:

PLATFORM_LINUX
PLATFORM_DARWIN
PLATFORM_WINDOWS
PLATFORM_BSD
PLATFORM_SOLARIS
PLATFORM_IOS
PLATFORM_ANDROID

We provide an enumeration for the operating systems:

namespace platform {
enum OS {
Linux,
Darwin,
Windows,
BSD,
Solaris,
IOS,
Android
};
}

Using constant expression we can export some operating system specific constants at compile time:

namespace platform {
constexpr OS os = Linux;
constexpr char sep = '/';
constexpr char devnull = '/dev/null';
}
